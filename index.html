What I've Built

inputX — Activity & Performance Monitor
2026 Python PySide6 (Qt) psutil pynput PyInstaller

inputX is a real-time desktop dashboard built for Windows that gives you a single, live view of how your PC and your inputs are performing — all in one lightweight app that runs quietly in your system tray. It visualizes CPU, RAM, disk, GPU, network, and every keypress you make, in a clean dark UI.

// Features
Live CPU, RAM, Disk & GPU usage with rolling history charts
Real-time keyboard heatmap — every keypress tracked & visualized
Network monitor — download/upload speed, ping & local IP
Mouse analytics — clicks, scroll distance & movement tracking
Controller page — connected gamepad input visualization
Game detection — auto-recognizes 30+ popular games from processes
System tray integration — minimize to tray, quick page-jump menu
Launch on startup toggle (Windows) in Settings
// Requirements
# System requirements Python 3.10+ | Windows 10/11 | VS Code (recommended)
// Setup Commands
# 1. Clone & open in VS Code git clone https://github.com/YOUR_USERNAME/inputx.git && cd inputx && code .
# 2. Create virtual environment python -m venv venv
# 3. Activate venv (Windows PowerShell) venv\Scripts\Activate.ps1
# 4. Install dependencies pip install -r requirements.txt
# 5. Run the app python main.py
# Optional: Build a standalone .exe pip install pyinstaller && pyinstaller --noconsole --onefile --icon=inputx_logo.png --add-data "inputx_logo.png;." main.py

MuteMaster — Discord Voice Bot
2026 Python discord.py

MuteMaster started as a voice-activity bot — it detects when someone mutes or deafens themselves (AFK), announces it in a chosen channel, and posts a summary of each voice session (duration & participants) once the channel empties out. It's since grown into a full Dead by Daylight session manager: a single button panel that handles killer rotation, ready checks, random perk builds, match results, and a live scoreboard — no typing commands required.

NovaCore Suite — All-in-One Discord Bot
2026 JavaScript Node.js discord.js Steam Web API

NovaCore Suite is an all-in-one Discord bot: custom welcome messages, a chat & reaction leveling system, a separate voice-activity leveling system, and a member-facing Steam stats suite. Everything runs off a local data file — no database setup required — so it installs cleanly on Windows and shared hosts alike.

// Features
Custom welcome embeds posted to a channel you pick, with placeholders
Chat & reaction leveling with anti-spam cooldowns — /level, /leaderboard
Separate voice-activity leveling track — /voicelevel, /voiceleaderboard
Steam stats suite — link an account and look up anyone's profile, games & achievements
Admin setup slash commands for welcome & level-up channels
Zero database — data persists to a local JSON file automatically
// Setup Commands
# 1. Download & extract novacore-bot.zip (link below), then open the folder cd novacore-bot
# 2. Configure your bot token, client ID & Steam API key cp .env.example .env
# 3. Install dependencies npm install
# 4. Register the slash commands with Discord npm run deploy
# 5. Start the bot npm start
